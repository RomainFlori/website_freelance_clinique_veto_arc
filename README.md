# 🐾 Clinique Vétérinaire de l'Arc

> Une expérience numérique moderne, fluide et rassurante dédiée au bien-être animal.

Ce projet est une plateforme web vitrine développée pour la **Clinique Vétérinaire de l'Arc**. L'objectif est d'allier une interface utilisateur (UI) luxueuse et épurée à une performance technique irréprochable, reflétant l'expertise médicale et la bienveillance de l'équipe.

---

## ✨ Caractéristiques du Projet

* **Design Glassmorphism :** Utilisation de flous d'arrière-plan et de transparences pour un look "Apple-style" moderne.
* **Expérience Fluide :** Animations lentes (`1000ms`), transitions "beurre fondu" et scroll doux (Smooth Scroll).
* **Architecture Mobile-First :** Entièrement responsive, optimisé pour une consultation rapide sur smartphone.
* **SEO Ready :** Génération automatique de Sitemap, métadonnées dynamiques et conformité RGPD (No-Cookie design).
* **Performance :** Optimisation des images via le format WebP et chargement différé (Lazy Loading).

---

## 🛠 Technologies Utilisées

Le projet repose sur une stack technologique de pointe (2026) :

| Technologie | Usage |
| :--- | :--- |
| **Nuxt 3** | Framework Vue.js haute performance avec rendu hybride (SSR/Static). |
| **Tailwind CSS** | Framework utilitaire pour un design sur mesure et ultra-léger. |
| **Nuxt Image** | Optimisation intelligente et redimensionnement automatique des visuels. |
| **TypeScript** | Typage statique pour un code robuste et maintenable. |
| **Vite** | Serveur de développement ultra-rapide et bundling optimisé. |

---

## 📂 Structure des Sections

1.  **Navigation :** Barre flottante avec animations de soulignement progressif au survol.
2.  **Hero Section :** Image de fond immersive avec effet de superposition textuelle et "boîte de verre".
3.  **Expertises :** Grille interactive présentant les services (Radiologie, Chirurgie, Ostéopathie, etc.).
4.  **Équipe :** Portraits dynamiques avec effets de soulèvement et badges de spécialité.
5.  **Contact & Accès :** Intégration Google Maps stylisée (Filtres CSS) et coordonnées détaillées.
6.  **Légal :** Pages conformes (RGPD, Mentions légales) avec un engagement "Zéro Cookie".

## 🚀 Installation & Développement

Assurez-vous d'avoir [Node.js](https://nodejs.org/) installé (version 18+ recommandée).

1.  **Cloner le dépôt :**
    ```bash
    git clone [https://github.com/votre-compte/clinique-arc.git](https://github.com/votre-compte/clinique-arc.git)
    cd clinique-arc
    ```

2.  **Installer les dépendances :**
    ```bash
    npm install
    ```

3.  **Lancer le serveur de développement :**
    ```bash
    npm run dev
    ```
    L'application sera disponible sur `http://localhost:3000`.

4.  **Générer le site pour la production :**
    ```bash
    npm run generate
    ```
